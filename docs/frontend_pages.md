# Frontend Pages

- Dashboard
- Strategies
- StrategyDetail
- GenerationRuns
- BacktestRuns
- BacktestTasks

Phase 0 页面使用 mock 数据和空状态，只验证路由、布局和信息结构。

## 页面边界

前端展示 AI 策略研发管理视图，不重复开发 FreqUI 已有运行态能力。需要查看 Freqtrade 运行界面时，通过 FreqUI 快捷入口跳转。

## Phase 1 页面目标

Phase 1 页面将接入策略列表、策略版本、生成批次、回测批次、回测任务、回测结果和策略排行榜 API。
